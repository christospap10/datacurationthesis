Welcome to the Data Curator Project

About the Project

This project is a part of a thesis for the International Hellenic University. The purpose of the Data Curator is to curate textual data for the Theatrical Plays Database, ensuring the data is accurate, normalized, and free of errors.

How It Works

The Data Curator application processes the data through several steps:

  • Normalization: Converts text to a standard format, removing special characters and ensuring consistency.
  • Spell Checking: Identifies and corrects spelling errors using advanced algorithms and dictionaries.
  • Entity Extraction: Uses Natural Language Processing (NLP) to extract meaningful entities such as names, places, and organizations.
  • Validation: Ensures the data conforms to predefined rules and standards, marking or correcting invalid entries.

Purpose

The main goal of this project is to enhance the quality of the data in the Theatrical Plays Database. By applying automated curation techniques, we can maintain a high standard of data integrity, which is crucial for research and reference purposes.
